## Communication Protocols

### operations-manager
- **Language:** Bahasa Indonesia (formal register)
- **Pronouns:** Use "Anda" (formal), never "kamu" (informal)
- **Suggestions:** Frame as "Mungkin bisa..." (perhaps we could...), not directives
- **Channels:** Slack for urgent, Dashboard for routine

### warehouse-manager
- **Language:** Chinese (Mandarin)
- **Channel:** Dashboard notifications only
- **Style:** Clear, concise operational instructions

### spiritual-director
- **Channel:** Email only — no Slack, no Dashboard
- **When to contact:** Buddhist practice questions, lineage content review, cultural sensitivity review, product authenticity verification
- **Response time:** Allow 48-72 hours

### mexico-fulfillment
- **Channel:** Email only
- **Language:** Spanish
- **Scope:** Latin American order fulfillment exclusively

## Role Resolution

Agents resolve roles via `ORG.md`. When a skill says "escalate to operations-manager", the agent:
1. Looks up the role in ORG.md
2. Uses the specified language and channel
3. Formats the message according to communication protocols

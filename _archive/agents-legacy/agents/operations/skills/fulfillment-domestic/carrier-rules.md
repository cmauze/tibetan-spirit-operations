# Carrier Selection Rules

## Default Carrier Matrix

| Weight | Dimensions | Value | Customer Choice | Carrier | Service |
|--------|-----------|-------|-----------------|---------|---------|
| <1 lb | Standard | Any | Standard | USPS | First Class |
| 1-5 lb | Standard | Any | Standard | USPS | Priority Mail |
| 5-20 lb | Standard | Any | Standard | UPS | Ground |
| >20 lb | Any | Any | Standard | UPS | Ground (freight quote if >70 lb) |
| Any | Any | Any | Expedited | USPS Priority Express or UPS 2-Day | Whichever is cheaper |
| Any | Oversized | Any | Any | UPS | Ground |

## Special Handling Rules

### Fragile Items
- Singing bowls (all sizes): UPS Ground preferred (better handling than USPS)
- Statues over $100: UPS Ground + declared value insurance
- Statues over $500: UPS Ground + full insurance + signature required

### High-Value Orders
- Orders >$200 total: Add signature confirmation
- Orders >$500 total: Add full insurance + signature required

### PO Box Addresses
- USPS only (UPS/FedEx/DHL cannot deliver to PO Boxes)
- If carrier rules suggest UPS but address is PO Box, fall back to USPS Priority

## Cost Optimization

Always query Shippo for real-time rates before purchasing labels. The matrix above provides defaults, but if Shippo shows a significantly cheaper option (>15% savings) that meets the delivery window, prefer the cheaper option.

## Packaging Standards

| Product Type | Packaging |
|-------------|-----------|
| Incense | Padded mailer or small box |
| Malas | Small box with tissue wrap |
| Prayer flags | Poly mailer |
| Singing bowls | Custom box with foam insert, double-boxed for >8 inch |
| Statues | Custom box, bubble wrap, double-boxed for >$200 |
| Thangkas (rolled) | Cardboard tube, never fold |
| Offering sets | Box with dividers between components |

# SEO and Organic Search Skill

## Overview
Build long-term organic search visibility through keyword research, on-page optimization, technical SEO, and content strategy.

## Key Sections

### SEO Strategy and Goals
- Define organic search role in marketing strategy
- Set traffic and ranking targets
- Identify priority keywords and topics
- Plan content roadmap aligned with keywords
- Set content production timeline
- Define success metrics for SEO

### Keyword Research and Analysis
- Conduct comprehensive keyword research
- Identify high-value, lower-competition keywords
- Analyze search intent (informational, transactional, etc.)
- Build keyword groupings by theme
- Identify long-tail keyword opportunities
- Monitor keyword trend changes

### Content Strategy and Planning
- Plan content calendar aligned with keywords
- Develop blog content strategy
- Create product content roadmap
- Plan category pages and taxonomy
- Identify content gaps and opportunities
- Coordinate with social-content team

### On-Page Optimization
- Optimize page titles and meta descriptions
- Structure content with proper headings
- Implement keyword optimization naturally
- Build internal linking strategy
- Optimize images with alt text
- Improve page load speed and mobile experience

### Technical SEO
- Ensure site crawlability and indexing
- Implement proper URL structure
- Fix crawl errors and broken links
- Implement XML sitemap
- Set up robots.txt properly
- Monitor Core Web Vitals and page speed

### Content Creation and Publishing
- Create high-quality, original content
- Follow SEO best practices in content
- Publish content consistently
- Update older content for freshness
- Repurpose content across formats
- Build content library and archives

### Link Building Strategy
- Develop link building outreach plan
- Build relationships with relevant sites
- Create linkable assets (guides, data, tools)
- Monitor backlink profile
- Disavow toxic or spammy links
- Implement internal linking strategy

### Local and International SEO
- Implement local SEO elements if applicable
- Optimize for local keywords and search
- Manage local business listings
- Plan international targeting if applicable
- Implement hreflang tags for multi-language
- Optimize for regional variations

### Competitive Analysis
- Analyze competitor keyword strategies
- Benchmark rankings and traffic
- Identify competitor content gaps
- Analyze competitor link profiles
- Identify competitive opportunities
- Monitor competitor SEO activity

### Performance Monitoring and Reporting
- Track keyword rankings and trends
- Monitor organic traffic and sources
- Analyze page performance metrics
- Track conversion rate from organic
- Calculate organic revenue and ROI
- Generate monthly SEO reports

### Continuous Optimization
- Monitor search trends and algorithm changes
- Adjust strategy based on performance
- Refresh and update underperforming content
- Optimize underperforming pages
- Expand on high-performing content
- Test new content formats

## Dependencies
- Integrates with: campaign-architecture, content-performance, social-content

## Success Metrics
- Organic traffic volume
- Keyword ranking positions
- Organic conversion rate
- Organic revenue and ROI
- Click-through rate from search results
- Top pages and landing pages

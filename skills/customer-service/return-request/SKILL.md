# Return Request Processing Skill

## Overview
Handle customer return and exchange requests following Tibetan Spirit return policy, generate RMA documentation, and manage the return logistics workflow.

## Key Sections

### Return Eligibility Check
- Verify purchase within return window (30 days)
- Check product condition and completeness
- Identify reason category (defective, wrong item, changed mind, etc.)

### RMA Generation
- Create unique RMA number
- Document reason and customer notes
- Generate return label or shipping instructions
- Set return condition expectations

### Return Authorization
- Communicate return address
- Explain refund or exchange timeline
- Clarify what items are returnable vs. non-returnable

### Exchange Processing
- Match customer requests to inventory
- Generate shipment for replacement
- Coordinate with fulfillment team

### Refund Coordination
- Verify return receipt
- Approve refund amount per policy
- Process refund to original payment method
- Document completion

## Dependencies
- Loads: product-knowledge for return eligibility rules
- Integrates with: inventory-management, fulfillment-domestic

## Success Metrics
- RMA generated and communicated
- Customer return tracking provided
- Refund or exchange initiated

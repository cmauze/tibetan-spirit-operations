# Review Solicitation Skill

## Overview
Systematically request customer reviews after purchase with channel-specific timing and messaging to maximize review collection and social proof.

## Key Sections

### Timing by Channel
- D2C/Shopify: Email 5 days post-delivery
- Etsy: Integrated Etsy messaging at optimal moment
- Amazon: Coordinate with FBA notification timing
- eBay: Follow eBay review request rules

### Customer Segmentation
- Filter for delivered orders only
- Exclude returns, disputes, or unhappy customers
- Target high-value or repeat customers first

### Review Request Messaging
- Channel-appropriate templates (tone, length, link type)
- Emphasize authenticity (genuine feedback requested)
- Offer incentive (discount code for future purchase) if appropriate
- Include product-specific value proposition

### Campaign Management
- Track request send dates and customer segments
- Monitor response rates and conversion to review
- A/B test messaging to improve response

### Review Management
- Monitor incoming reviews for negative feedback
- Flag for response or escalation if needed
- Highlight positive reviews for marketing use

## Dependencies
- Integrates with: fulfillment tracking, order-inquiry
- Loads: channel-config for platform-specific rules
- References: product-knowledge for product highlights

## Success Metrics
- Review request send rate
- Review submission rate
- Review rating average and distribution

# Fulfillment - Nepal Operations Skill

## Overview
Coordinate inbound supply chain from Nepal suppliers through customs clearance and into US inventory, managing documentation and regulatory compliance.

## Key Sections

### Supplier Purchase Orders
- Coordinate with supplier-communication skill
- Monitor PO status and shipment timeline
- Track expected arrival dates

### Inbound Shipment Tracking
- Monitor shipments from Kathmandu
- Track transit milestones
- Identify delays or issues

### Customs Documentation
- Verify supplier commercial invoices
- Assign HS (Harmonized System) codes to products
- Determine landed costs and duties
- Prepare customs entry documents

### HS Code Management
- Reference HS code classification rules
- Apply codes by product category (incense, statues, textiles, etc.)
- Track code changes or tariff updates
- Maintain HS code library for consistency

### Inbound Quality Check
- Verify goods match PO specs on arrival
- Document any damage or discrepancies
- Process damage claims if needed

### Inventory Integration
- Route received goods to appropriate fulfillment location
- Update inventory management system
- Log receipt and QC results

## Dependencies
- Integrates with: supplier-communication, inventory-management
- References: product-knowledge for SKU classification

## Success Metrics
- On-time customs clearance
- Accurate HS code application
- Inventory received and recorded

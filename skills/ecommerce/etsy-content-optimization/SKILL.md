# Etsy Content Optimization Skill

## Overview
Maximize Etsy search visibility through strategic optimization of titles, tags, descriptions, and categories using Etsy-specific SEO best practices.

## Key Sections

### Etsy Title Optimization
- Research Etsy search terms and keywords
- Craft titles with primary keyword first (140 char limit)
- Include product type, material, and attributes
- Balance keywords with readability
- Test title variations for performance
- Update underperforming titles

### Etsy Tag Strategy
- Build tag library for products (13 tags maximum per listing)
- Research high-volume, low-competition tags
- Include product type, style, material tags
- Add buyer intent and use-case tags
- Monitor tag performance data
- Refresh tags seasonally or based on trends

### Description Optimization
- Write compelling product descriptions
- Include relevant keywords naturally
- Address customer pain points and benefits
- Use clear formatting and structure
- Add material, dimensions, care instructions
- Include shipping and customization info

### Photo and Visual Optimization
- Ensure first image is eye-catching and clear
- Follow Etsy photo size and quality specs
- Showcase product from multiple angles
- Include lifestyle shots when appropriate
- Update photos for seasonal relevance
- Monitor photo performance metrics

### Category and Attribute Selection
- Assign correct Etsy category
- Complete all required attributes
- Use attributes customers actually search
- Optimize for category-specific visibility
- Avoid attribute errors that hurt visibility

### Listing Monitoring and Iteration
- Track listing search impressions and position
- Monitor click-through rates
- Analyze conversion by listing
- Identify underperforming listings
- Test optimization changes
- Document what works

### Seasonal Optimization
- Align keywords with seasonal demand
- Update titles/tags for holidays and events
- Manage seasonal product visibility
- Plan content updates by season

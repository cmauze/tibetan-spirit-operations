# Site Health Monitoring Skill

## Overview
Continuously monitor Shopify storefront uptime, performance, errors, and user experience to ensure optimal operation and identify issues quickly.

## Key Sections

### Uptime Monitoring
- Monitor site availability and response time
- Track server status and errors
- Alert on outages or slowdowns
- Verify SSL certificate and security status

### Performance Metrics
- Monitor page load times
- Track Core Web Vitals (LCP, FID, CLS)
- Identify slow pages or resources
- Monitor server response times

### Error Tracking
- Monitor application error rates
- Track 4xx and 5xx HTTP errors
- Identify broken links or pages
- Alert on critical errors

### Checkout Monitoring
- Track checkout funnel health
- Monitor payment processor connections
- Verify order submission success rate
- Alert on checkout failures

### Third-Party Integration Health
- Monitor payment gateway status
- Verify email delivery systems
- Check shipping carrier API health
- Monitor analytics tracking

### User Experience Monitoring
- Track visitor traffic patterns
- Monitor bounce rates and session duration
- Identify pages with usability issues
- Gather user feedback signals

### Performance Reporting
- Generate daily health reports
- Track metrics over time
- Identify trends and patterns
- Recommend optimizations

### Incident Response
- Alert team on critical issues
- Document incidents and resolutions
- Implement fixes and monitors
- Post-incident reviews

## Dependencies
- Integrates with: content-performance, drift-detection
- Monitors: Shopify platform, third-party services

## Success Metrics
- Site uptime percentage
- Average page load time
- Error rate
- Checkout conversion rate

# Collection Management Skill

## Overview
Strategically organize products into Shopify collections to improve discovery, merchandising, and customer experience through thoughtful curation.

## Key Sections

### Collection Planning and Strategy
- Identify collection themes and purposes
- Plan seasonal collections aligned with tibetan-calendar
- Design bundles for cross-selling and upsells
- Organize by product type, use case, or customer segment
- Align with marketing campaigns and promotions

### Collection Creation
- Create Shopify collections with SEO titles
- Write compelling collection descriptions
- Design collection imagery and themes
- Set collection sorting and filtering rules
- Publish and promote new collections

### Seasonal Collections
- Plan collections for Losar, Vesak, and seasonal events
- Curate products relevant to seasonal practices
- Update collections with new seasonal products
- Archive collections when no longer relevant
- Prepare next season's collections in advance

### Promotional Collections
- Create collections for marketing campaigns
- Bundle related products for promotion
- Set up limited-time promotional collections
- Coordinate with campaign-architecture skill
- Track collection performance during promotions

### Product Grouping and Discovery
- Organize by beginner/intermediate/advanced level
- Create practice-type collections (meditation, ritual, daily)
- Build material and type collections
- Create gift collections and bundles
- Organize by price point and value tier

### Collection Performance Monitoring
- Track collection traffic and conversion
- Monitor click patterns within collections
- Analyze which products are viewed together
- Identify underperforming collections
- Optimize collection order and layout

### Collection Maintenance
- Review and refresh collection content regularly
- Update product descriptions within collections
- Remove discontinued products
- Add new relevant products
- Monitor for seasonal relevance
- Retire outdated collections

### Search and Navigation Optimization
- Optimize collection navigation structure
- Create clear hierarchy and menu structure
- Enable proper filtering and faceting
- Improve discoverability through internal linking
- Test collection findability

## Dependencies
- Integrates with: content-performance, cross-channel-parity, tibetan-calendar
- References: product-knowledge for product organization

## Success Metrics
- Collection traffic volume
- Collection conversion rate
- Average order value from collection sales
- Product views per collection
- Collection creation frequency
